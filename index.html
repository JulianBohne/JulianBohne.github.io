Navigation

The basic slide interface allows the following actions:

  • The cursor-left- and cursor-right-keys can be used to move through the slides.
  • The cursor-up- and cursor-down-keys can be used to navigate to subslides, if available.
  • Click on the Burger-Menu-button (top left) to open the navigation menu
  • Click on the Questionmark button (top right) to ask questions and give feedback about the current slide.
  • Click on the Play button (top right) to start the video recording of the presentation on the current slide, if available.
  • CTRL+F or CMD+F opens a search dialog.
  • You can close all open menus, dialogs or videos with the ESC key.

Interaction

Within the slides you may encounter interactive elements. You can interact with them in the following ways:

  • You can zoom in on each element by double-, tripple- or alt-clicking, depending on the slide's settings.
  • A slide with additional whiteboard-pages will have a blinking bottom border. You can scroll down on these slides to view the additional content.
  • Eingebettete Demos und Webseiten k√∂nnen Sie mit der Expandieren Schaltfl√§che in den Vollbildmodus versetzen.
  • Embedded demos and websites can be expanded to fullscreen with the Expand button.

Navigation Menu

The navigation menu contains the following features:

  • Jump to any slide.
  • Open the search dialog (keyboard shortcut CTRL+F).
  • Open slides in print view.
  • Switch between light- and dark-mode.
  • Toggle the Handout-View of the slides (keyboard shortcut H-H-H).
  • Toggle the Presenter-Mode of the slides (keyboard shortcut P-P-P).
  • Initialize the video recording (keyboard shortcut R).
  • Toggle contrast colors and other accessibility features (keyboard shortcut A-A-A).

Feedback Menu

The questions and feeback menu can be used to comment on the current slides:

  • You can submit your entered questions by pressing Shift + Enter or by pressing the Send button.
  • Sie k√∂nnen Fragen, die Sie beantwortet sehen wollen mit der Daumen Hoch -Schaltfl√§che bef√ºrworten.
  • You can up-vote questions that you want to see answered with the Thumbs Up button.
  • You can edit your own questions with the Edit button.
  • You can delete your own questions with the Trashcan button.

Other keyboard shortcuts

There are even more keyboard shortcuts that you can use while browsing the slides:

  • The F key activates fullscreen mode. You can leave it with the ESC key.
  • The ? key opens a cheatsheet for all available keyboard-shortcuts.

Keyboard shortcuts during recording playback

When you are viewing the presentation's recording you can use the following keyboard shortcuts:

  • ESC: Closes the video and moves to the slide currently visible in the video.
  • Space/K: Pauses the video.
  • J/L: Jump 10 seconds back or ahead.
  • Cursor-left/Cursor-right: Jump to the previous or next slide in the video.
  • Cursor-up/Cursor-down: Increase or decrease volume.
  • M: Mute/unmute.

On Smartphone or Tablet:

  • Double Tap the left or right edge of the screen: Jump 10 seconds back or ahead.
  • Double Tap the center of the screen: Pauses the video.

Additional information for visually impaired users

The following hints may be useful for users of screenreaders:

  • The first element in tab-order is the navigation menu.
  • The last elements in tab-order are the buttons to navigate to the previous or next slide.
  • The slide-number between the previous and next buttons is a link without further function, so it can be used to quickly navigate to these buttons.
  • The navigation menu has three tab-groups: The close button, the action buttons and the slide list. Within these groups you can select different elements with the cursor keys.
  • Within the feedback menu, if there are questions available, you can tab into the list of questions and navigate withing the questions with the following keys:
    • Navigate between questions with the cursor keys.
    • Interact with a question with the Enter key.
    • Return to the question list with the ESC key.
  • Within the search dialog you can press Enter to jump to the slides the entered keyword was found.
  • Math formulas are rendered using the MathJax library. You can interact with them with the following keys when the accessibility features are enabled:
    • Space opens a menu to change the settings of your assistive view.
    • Enter opens a dialog that can help you navigate through complex fomulas and equations with the cursor keys.

If using a screen reader we recommend the usage of the handout mode:

  • The usual presentation view only allows assistive technology to see the current slide.
  • The handout view allows assistive technology to see the entire contents of the slides at once. You can especially use the headings of slide titles for navigation.
